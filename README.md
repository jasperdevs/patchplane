# Patchplane

A taste-first, agent-native software forge concept.

Patchplane is an early product and brand foundation for a GitHub alternative that treats source hosting, review, CI, and agent context as one coherent developer workspace.

## What is here

- `src/app` - a monochrome Next prototype using shadcn-compatible tokens.
- `docs/research` - market notes and video-derived analysis.
- `docs/brand` - naming, positioning, visual standards, and the initial shadcn theme.

## Local setup

```bash
npm install
npm run dev
```

## Product thesis

GitHub won the last generation by making open source feel social and default. The next forge needs to preserve that network feeling while solving trust, speed, review quality, and agent-era history.
